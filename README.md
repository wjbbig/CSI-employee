# CSI-employee
基于SSM的员工后台管理系统   
前端使用layui框架,项目整体使用Spring+SpringMVC+Mybatis，权限模块使用RBAC权限模型。  
数据库使用Mysql，部署使用Tomcat。 项目默认登录账密：superadmin/123456
